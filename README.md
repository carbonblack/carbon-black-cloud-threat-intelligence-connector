# STIX Threat Connector for CBC

This is a python project that can be used for ingesting Threat Intelligence from various STIX Feeds. The current supported versions of STIX Feeds are:

| STIX Version| Supported   |
| ----------- | ------------|
| 1.0.1       | No          |
| 1.1         | Yes         |
| 1.2         | Yes         |
| 2.0         | Yes         |
| 2.1         | Yes         |
